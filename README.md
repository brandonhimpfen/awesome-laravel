## Official Resources

- [Laravel.com](https://laravel.com/) – Official documentation.
- [Laravel API Docs](https://laravel.com/api/) – Full API reference.
- [Laravel GitHub](https://github.com/laravel)
- [Laravel News](https://laravel-news.com/) – Articles, packages, podcasts.
- [Laracasts](https://laracasts.com/) – Premier educational platform for Laravel.

## Starter Kits & Boilerplates

- [Laravel Breeze](https://laravel.com/docs/starter-kits#laravel-breeze) – Simple auth scaffolding.
- [Laravel Jetstream](https://jetstream.laravel.com/) – App scaffolding with Teams, 2FA, Sessions.
- [Laravel UI](https://github.com/laravel/ui) – Legacy UI scaffolding.
- [Laravel Zero](https://laravel-zero.com/) – Build console applications using Laravel components.
- [Wave](https://github.com/thedevdojo/wave) – SaaS starter kit built on Laravel.

## Authentication & Authorization

- [Laravel Fortify](https://github.com/laravel/fortify) – Backend authentication services.
- [Laravel Sanctum](https://laravel.com/docs/sanctum) – API tokens & SPA authentication.
- [Laravel Passport](https://laravel.com/docs/passport) – OAuth2 server implementation.
- [Spatie Laravel Permission](https://github.com/spatie/laravel-permission) – Role & permission management.
- [JWT Auth](https://github.com/tymondesigns/jwt-auth) – JSON Web Token authentication.

## APIs, HTTP & GraphQL

- [Laravel JSON:API](https://github.com/laravel-json-api/laravel) – JSON:API server + client implementation.
- [Laravel API Resources](https://laravel.com/docs/eloquent-resources)
- [Folklore GraphQL](https://github.com/Folkloreatelier/laravel-graphql) – GraphQL server implementation.
- [Rebing GraphQL](https://github.com/rebing/graphql-laravel) – GraphQL library for Laravel.
- [Laravel OpenAPI / Swagger Generators](https://github.com/DarkaOnLine/L5-Swagger) – Auto-generate API docs.

## Database & ORM

- [Eloquent ORM](https://laravel.com/docs/eloquent) – Intuitive active-record ORM.
- [Laravel Scout](https://github.com/laravel/scout) – Full-text search for Eloquent.
- [Laravel Telescope: Query Watching](https://laravel.com/docs/telescope)
- [Spatie Laravel Query Builder](https://github.com/spatie/laravel-query-builder) – Fluent API for filtering, sorting & includes.
- [Blueprint](https://github.com/laravel-shift/blueprint) – Generate models, migrations & controllers from YAML.

## Queues, Jobs & Events

- [Laravel Horizon](https://laravel.com/docs/horizon) – Queue dashboard for Redis.
- [Laravel Queue](https://laravel.com/docs/queues)
- [Laravel Event System](https://laravel.com/docs/events)
- [Laravel WebSockets](https://github.com/beyondcode/laravel-websockets) – Pusher-compatible WebSocket server.

## Security

- [Laravel Security Docs](https://laravel.com/docs/security)
- [Laravel Security Checker](https://github.com/enlightn/security-checker) – Check dependencies for vulnerabilities.
